Themenmodule
Globalgeschichte als Lernpfade
Jedes Modul übersetzt ein Thema der Globalgeschichte in einen Lernpfad, der an RZG-Kompetenzen ausgerichtet ist.

Architektur des Prototyps

Der Prototyp läuft vollständig im Browser (keine Installation). Die Datenebene ist als austauschbares Modul gebaut: für die Pilotierung im grösseren Massstab kann sie durch ein Backend (z. B. React + Datenbank) ersetzt werden.
