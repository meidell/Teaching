Set up the game

Team game, run from the board (Zoom / screen share). Add 2 to 5 teams, then start the game.

Rules
  • The team whose turn it is picks a tile (category + points).
  • You read the question aloud. The team answers before the timer runs out (30 s).
  • Click Reveal, then award points: Correct (active team) or open the steal to let another team try.
  • Correct answer = +points. A successful steal = +points for the stealing team.
  • The game ends when all tiles are played. Highest score = winner.
